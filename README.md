# Asha AI Chatbot - MVP

A context-aware, women-first AI chatbot built for the JobsForHer platform.

## MVP Goals
- Multi-turn, contextual conversations
- Career-focused guidance for 3 cohorts:
  - Starters
  - Restarters
  - Risers
- Bias detection & redirection
- Session-based memory (no PII)
- Retrieval-Augmented Generation (RAG)
- Emotion-aware and empathetic replies
- Guardrails and secure responses

## Tech Stack
- Python (LangChain, OpenAI)
- Streamlit for UI
- FAISS for vector search
- Firebase (optional) for feedback storage
